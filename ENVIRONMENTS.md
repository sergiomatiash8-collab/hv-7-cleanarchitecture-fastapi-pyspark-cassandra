Development (Dev)
What: laptop, local testing
Features:
  - Debug logging (verbose)
  - Hot reload (code changes auto-reload)
  - Mock data (fake reviews for testing)
  - Local database (on your machine)
  - No SSL/HTTPS (just http://localhost)
  - Fast startup

Production (Prod)
What: Real server, real users
Features:
  - Error logging only (no debug spam)
  - No hot reload (stability matters)
  - Real data (actual customer reviews)
  - Remote database (cloud or server)
  - SSL/HTTPS (secure)
  - Performance optimized
